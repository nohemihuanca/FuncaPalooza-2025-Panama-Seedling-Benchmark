# FuncaPalooza 2025 Panama Seedling Benchmark

## Overview

This benchmark contains 605 curated observations (~3,000 images) of tropical woody seedlings from lowland forests of Panama.

The dataset was assembled from the SeedLearn Project to support species identification, computer vision, and biodiversity informatics research.

## Dataset Access

Images and observation metadata are hosted through the iNaturalist project:

https://www.inaturalist.org/projects/funcapalooza-2025-panama-seedling-benchmark

A frozen snapshot of benchmark observations and labels is provided in:

[benchmark_observations.csv](benchmark_observations.csv)

## Taxonomic Labels

Benchmark labels correspond to expert determinations provided by project curator Nohemi Huanca-Nunez.

For benchmarking purposes, the most recent identification provided by Nohemi Huanca-Nunez should be considered the reference taxonomic label.

Because iNaturalist community identifications evolve through time, community taxa may differ from the benchmark labels used in this dataset.

## Dataset Characteristics

- 605 observations
- Approximately 3,000 images
- 34 species (Version 1.0)
- Multiple photographic views per individual seedling
- Tropical forests of Panama

## Source Project

This benchmark was derived from observations collected as part of the SeedLearn Project.
